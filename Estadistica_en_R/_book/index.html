Chapter 1 Acerca de este libro

Este libro se creó para funcionar como una guía de trabajos prácticos y una introducción al manejo del software de R para estadística básica. Forma parte integral del curso de estadística bàsica para las neurociencias dictado por Fleni. Para más información sobre este curso por favor visitar el siguiente link https://www.fleni.org.ar/investigacion-educacion/cursos-seminarios-y-congresos/
